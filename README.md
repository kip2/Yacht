# ヨット

シンプルなダイスゲーム「ヨット」をブラウザで遊べるアプリ。
プレイヤー 1〜5 人、サーバ不要、ブラウザだけで完結します。

**プレイ**: <https://kip2.github.io/Yacht/>

(日本語 / English の切替は右上のボタン。「遊び方」ボタンからルールを参照できます。)

## 遊び方

5 個のサイコロを振り、12 のカテゴリそれぞれに点数を記入していく。
全カテゴリが埋まった時点で **合計点が最も高い人の勝ち**。

### 1 手番の流れ

1. **「振る」** で 5 個全部のサイコロを振る (手番の最初の 1 回)。
2. 残したいダイスをクリックで選び、**「選んだダイスを振り直す」** で
   選んだダイスを振り直す。1 手番に振れるのは合計 3 回まで。
3. まだ使っていないカテゴリを 1 つ選んで **点数を確定** (0 点でも記入可)。
   確定したら次のプレイヤーの手番に進む。

### 役一覧

| 役                | 条件 / 点数                                              |
| ----------------- | -------------------------------------------------------- |
| エース 〜 シックス | その目の出目の合計 (例: エース で `[1,1,2,3,4]` → 2 点) |
| フルハウス         | 3 個揃い + 2 個揃い → 5 個の合計 (例: `[3,3,3,2,2]` → 13) |
| フォーダイス       | 同じ目 4 個以上 → その目 × 4 (例: `[4,4,4,4,1]` → 16)    |
| リトルストレート   | 1-2-3-4-5 が揃う → 30 点                                  |
| ビッグストレート   | 2-3-4-5-6 が揃う → 30 点                                  |
| チョイス           | 5 個の合計 (制約なし、捨て役にも使える)                   |
| ヨット             | 5 個すべて同じ目 → 50 点                                  |

ゲーム画面の「遊び方」ボタンからも同じ説明が確認できます。

## ローカルで動かす

クローン後、いずれか好きな方法で動作確認できます。

### A. お手軽 (Python があればそれだけ)

```sh
scripts/serve.sh                # → http://127.0.0.1:8765/
```

リポジトリにコミットされている `public/yacht.wasm` をそのまま配信します。
JS / HTML / CSS だけ編集してブラウザで `Ctrl+Shift+R` すれば反映されます。

### B. Docker

ローカルに D 言語環境を入れたくない場合:

```sh
docker compose up --build       # → http://127.0.0.1:8765/
```

コンテナ内で WASM を再ビルドしてから配信するので、`source/` を編集して
`docker compose up` し直せば反映されます。停止は `Ctrl+C`、片付けは `docker compose down`。

### C. ローカル D 環境で再ビルド

D ソースを編集して WASM を更新したい場合 (詳細は `docs/README.md`):

```sh
sudo pacman -S ldc lld python   # 一度だけ (Arch の場合)
scripts/build-wasm.sh            # → public/yacht.wasm を更新
scripts/serve.sh                 # → http://127.0.0.1:8765/
```
